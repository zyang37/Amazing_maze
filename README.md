# **Amazing Maze**
### Run
- install Pygame  
    sh setup.sh
- Have fun  
    python3 start.py

### Current Progress
- Moving the character
- Generating a random maze
- A randomly appear star
- Pick up the star!
  - Get a higher score
  - Destroy a random wall
- Added enemies
- 4 Difficulty levels
- Added a Start Menu
